# gateway-fe

Frontend scaffold for the gateway profile automation platform.

## Stack

- React
- TypeScript
- Vite

## Screens included

- Overview dashboard
- Profile management
- Proxy management
- API key management
- Automation jobs and Playwright preview

## Quick start

```bash
cp .env.example .env
npm install
npm run dev
```

Set `VITE_API_URL` to the backend origin, for example `http://localhost:8000`.
